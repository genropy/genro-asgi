About genro-asgi

genro-asgi is a lightweight ASGI server framework designed for simplicity and performance.

Why genro-asgi?

  • Zero external dependencies (stdlib only)
  • Full type hints for better IDE support
  • Simple TOML-based configuration
  • Built-in static file serving
  • Middleware support

Getting Started

# Install
pip install genro-asgi

# Create config.toml
[server]
host = "127.0.0.1"
port = 8000

[staticsites.main]
path = "/"
directory = "public"

# Run
genro-asgi serve .
